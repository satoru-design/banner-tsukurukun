# 広告バナーの王道デザイン要素 調査レポート

- **調査日**: 2026-04-21
- **目的**: banner-tsukurukun v2 Phase A.5（広告品質改善）の実装参考
- **調査対象**: design-library.jp 起点、Anagrams / LISKUL / LIG / Lany / BanaLabo / アーティス / 折兼 など日本語権威サイト 10 件

---

## 1. レイアウトの王道（3 層構造で整理）

### L1: 視線誘導パターン（最上位）
| パターン | 特徴 | 適性 | 頻度 |
|---|---|---|---|
| **Z 型** | 左上→右上→左下→右下の視線動線、中央配置でインパクト重視 | 一般消費財・EC | ★★★★★ |
| **F 型** | 左テキスト＋右画像、誠実感を演出 | BtoB・教育・情報系 | ★★★★☆ |
| **縦割り型** | 画面を垂直分割、同等重要度訴求 | 商品・ブランド訴求 | ★★★☆☆ |
| **フッター型** | 下部にテキストまとめて CTA 設置 | キャンペーン告知 | ★★★☆☆ |

### L2: 構成パターン（中位）
- **ベーシック型**: コピー・訴求文・写真をバランス良く、情報量多
- **キャッチコピー型**: 要素を削り、コピーで勝負
- **写真メイン型**: 商品の視覚的インパクト優先
- **バリエーション型**: 複数商品・カラーの並列
- **記事風型**: 広告感を消して視認性・情報量向上
- **診断型**: 「あなたはどのタイプ？」で誘導

### L3: 空間配置（下位）
- 横割り（上下分割、コピー+写真）
- 縦割り（左右分割）
- 敷詰め（画像多め）
- 全面配置（文字 or 画像を大きく）
- 斜め配置・余白活用（高級感）

## 2. コピー構造の 8 つの訴求型（アナグラム株式会社）

| # | 型 | 訴求軸 | 典型例 |
|---|---|---|---|
| 1 | **ターゲット絞込み型** | 問いかけ・呼びかけ | 「〇〇な人必見」「40代の方へ」 |
| 2 | **感情代弁型** | 共感・悩みの可視化 | 「また同じ失敗…」「なぜ私だけ？」 |
| 3 | **啓蒙型** | 気づき・現状批判 | 「その習慣、実は…」 |
| 4 | **比較型** | 差別化・相対評価 | 「Before→After」「他社比」 |
| 5 | **数字表現型** | 具体性・説得力 | 「1杯120円」「367種の栄養」 |
| 6 | **権威型** | 信頼・安心感 | 「売上No.1」「専門医推奨」 |
| 7 | **利用シーン提案型** | 具体的使用法 | 「ヘアケアにも使える美容オイル」 |
| 8 | **臨場感演出型** | 五感刺激 | 「とろ〜り」「さらさら」 |

**Phase A.5 への示唆**: 現在の 4 アングル（Benefit / Fear / Authority / Empathy）はこれら 8 型の上位抽象。上記 8 型を**サブ型**として JSON に追加すれば、より具体的なコピーが出る。

## 3. CTA ボタン ベストプラクティス

### 色選択
| 色 | 効果 | 適性 |
|---|---|---|
| **緑** | 「進む」のイメージ、健康・自然・エコ | 定番、健康食品・通販 |
| **オレンジ** | 緊急性・暖かみ、行動喚起 | 定番、EC全般 |
| **赤** | 緊急性・情熱・セール | 限定品・期間限定 |
| **青** | 信頼・安定 | 金融・医療・BtoB |
| **金** | 高級感・権威 | プレミアム商材 |

**原則**: 背景色と対照的な色を選ぶ。背景が白→緑/オレンジ、背景が黒→金/白。

### 文言パターン
- **動詞型**: 「今すぐ購入」「無料で試す」「登録する」
- **疑問型**: 「〇〇したいと思いませんか？」
- **時間限定型**: 「今だけ」「今日中に」「本日限り」
- **数量限定型**: 「残りわずか」「先着100名」（FOMO 心理）

**NG**: 「こちらをクリック」（何が得られるか不明）

### ボタン要素
- 背景と明確なコントラスト
- 角丸矩形 or カプセル型（モダン）
- **視線誘導の▶矢印**（参考バナーで高頻度）
- ドロップシャドウ・微アニメーションで浮き上がり感
- 人物写真の視線を CTA に向ける

---

## 4. 価格・訴求バッジ

### 文言パターン（使用頻度順）
1. **「初回購入限定 ¥980」** （D2C・サブスク定番）
2. **「50%OFF / 2,500円（税込）」** （セール定番）
3. **「送料無料」**
4. **「今だけお試し 1,000円」**
5. **「先着100名限定」**
6. **「早割キャンペーン」**
7. **「期間限定」**

### 形状
- **長方形**: どんなデザインにも合う
- **円形**: 親しみやすさ・セール感
- **カプセル型**: モダン・洗練（近年人気）
- **斜め配置**: 動きのあるインパクト
- **リボン型**: キャンペーン告知

### 配置の定番
- 画像の左下 or 右上（視線の終点 or 始点）
- 主被写体の近く（関連性で認知促進）

## 5. 文字装飾（座布団・フチ・シャドウ）

### 座布団（背景プレート）の使い分け
| 形状 | 印象 | 適性 |
|---|---|---|
| 長方形 | 整った・誠実 | BtoB・情報系 |
| 円形 | 柔らかい・かわいい | コスメ・食品 |
| カプセル型 | モダン・洗練 | SaaS・IT |
| 手書き風 | カジュアル | SNS・ブログ |
| ボタン風 | クリック誘導 | CTA・バッジ |

**色の指針**: 背景が暗→明るめ座布団 / 背景が明→濃い座布団
**NG**: 全要素に座布団を使うと画面散漫 → ポイント使用

### フチ取り
- **白フチ**: 背景が暗い・画像がカオス → 文字を浮き立たせる定番
- **黒フチ**: 背景が明るい・白文字を強調
- **二重フチ** (白+黒など): さらに強い可読性、ゲーム系・煽り系

### ドロップシャドウ
- コピー部分で可読性補助
- 強度控えめ（不自然に見えない範囲で）

## 6. ジャンプ率（文字サイズ比率）の具体数値

| サイト | 大見出し : 中見出し : 本文 | 印象 |
|---|---|---|
| 無印良品 | **3.6 : 1.57 : 1** | 賑やか・躍動的 |
| Amazon | **2.1 : 1.6 : 1** | 実用的・EC 向け |
| iPad Pro (Apple) | **2.0 : 1.25 : 1** | 高級・落ち着き |

### Phase A.5 推奨
- **広告バナーは高ジャンプ率**（3〜4 倍）が効く → 「スルッと-2kg」の数字は他の 3 倍近く
- 強調 1 単語のみ拡大、残りは揃える
- 「目立たせたい単語は思い切って 2 倍以上」（Anagrams 推奨）

## 7. 業種別の傾向（観察ベース）

| 業種 | 典型色 | レイアウト | 訴求軸 |
|---|---|---|---|
| **健康食品・サプリ** | 緑（自然） / 金（権威） | 左テキスト+右人物+商品 | 数字＋権威 |
| **美容・ヘアケア** | ピンク / 白（清潔） | センター or Z 型 | 感情代弁＋ベネフィット |
| **不動産** | 紺 / ベージュ | フッター型 | 数字＋権威 |
| **旅行** | 空・海の青 / 赤 | 全面配置 | ベネフィット＋価格 |
| **BtoB・SaaS** | 青 / 白 | F 型 | 権威＋利用シーン |
| **サブスク・D2C** | ブランド色 / オレンジ | Z 型 | 初回限定＋数字 |

## 8. NG パターン（やってはいけない）

1. **背景色と文字色の同化** → メッセージが伝わらない
2. **低解像度・ぼやけた画像** → 信頼低下
3. **装飾の全要素多用**（座布団+フチ+シャドウ+グラデ） → 散漫
4. **CTAが目立たない**（周囲と同系色） → CTR 低下
5. **コピーが 3 行以上の長文** → スキップされる
6. **曖昧な CTA**（「こちらをクリック」） → 何が起きるか不明
7. **低ジャンプ率の情報詰め込み** → どこを読めばいいか分からない
8. **ブランド色を無視した色選択** → 統一感なし

## 9. 一般ベンチマーク
- 日本の平均バナー CTR: **0.1〜0.3%**
- 高成果バナー: 1% 以上を目安

## 11. Phase A.5 への実装示唆（調査から導いた具体策）

### 必須で実装すべき（「広告らしさ」の決定打）
1. **コピー生成 JSON に価格バッジ項目追加**: `priceBadge: { text, type, style }`（例: `{ text: "初回限定 ¥980", type: "price", style: "circle-red" }`）
2. **CTA テンプレート 5 種**: 緑/オレンジ/赤/金/青の角丸矩形、矢印あり・なし版
3. **ジャンプ率の自動設計**: メインコピー内の「強調 1 単語」を他の 2〜3 倍サイズに自動設定
4. **座布団の自動判定**: 背景コントラスト解析 → 文字が埋もれるなら自動で座布団追加（Phase C と統合可）

### コピー生成プロンプトへの注入語
- 「直接訴求型（ダイレクトレスポンス）」
- 「数字+権威+緊急性の3層」
- 「日本の広告バナー定番フォーマット」
- 「強調1単語を決定し、それを2倍以上でジャンプさせる」

### 画像生成プロンプトへの注入語
- "Japanese direct response ad banner aesthetic"
- "product prominently featured, bold composition"
- "negative space designed for overlay typography"
- "commercial photography with dramatic lighting"
