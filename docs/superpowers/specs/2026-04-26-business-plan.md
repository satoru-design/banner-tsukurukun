- **LP**: `autobanner.jp/lp01`, `/lp02` ...（A/Bテスト前提のサブパス構成）

- middleware で `/lp**` は無認証、`/` 以下のアプリは Google SSO 必須

### 5.4 問い合わせフォーム
- 既存 Formspree (`xaqaazaz`) 流用 もしくは新規発行（free tier）

## 🆕 v2 補足: 完全却下機能リスト（実装しない明確化）

専門家提案20項目のうち、以下は**戦略的に却下**:

| 提案 | 却下理由 |
|---|---|
| **RL学習（Reinforcement Learning from Ad Performance）** | GPU費用爆発・学習データ著作権・運用工数すべて Affordable Loss 超過 |
| **競合URL スクレイピング解析** | 不正競争防止法/著作権法のグレーゾーン → 信頼毀損リスク高 |
| **CRDT 同期編集（マルチユーザーリアルタイム編集）** | データ消失時の賠償リスク >> 価格帯（¥14,800/月）に見合わない |
| **Meta OAuth 直結 CTR 連携** | 個人情報取扱の責任 vs 訴求効果薄。代替: CSV アップロード方式（Phase B.1 で検討） |
| **類似商標・意匠チェック機能** | 実装難・大手要望あれば個別対応で十分 |

これらは**Affordable Loss を超える失敗時損失**のため、現在のフェーズでは触らない。
